# objetivo 
Por favor abra apenas SistemaUFC.py

Este projeto implementa um sistema simples para simular a estrutura da Universidade Federal do Ceará (UFC), utilizando os conceitos de Programação Orientada a Objetos (POO) em Python com armazenamento em Listas.
O sistema permite gerenciar a relação de Um-para-Muitos, onde cada Campus pode ter vários Cursos.

# das funções 
- criar:
Cadastrar um novo Campus.
Adicionar um novo Curso a um Campus específico.

- ler:
Listar todos os Campi cadastrados.
Listar todos os Cursos de um Campus selecionado.

- atualizar:
Editar o nome e o número de vagas totais de um Curso existente.

- excluir:
Remover um Campus (o que exclui automaticamente todos os seus Cursos).
Excluir um Curso específico de um Campus.
